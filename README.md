Descrição da solução:
Durante missões espaciais longas, como viagens para Marte ou permanência em bases lunares, o transporte frequente de alimentos da Terra se torna inviável. Pensando nesse desafio, desenvolvemos o COSMOFARM, um sistema que simula o gerenciamento de uma estufa espacial.
O programa permite consultar informações das plantas cultivadas, monitorar sensores da estufa, visualizar alertas do sistema e calcular recursos necessários para missões espaciais. Dessa forma, o astronauta consegue acompanhar as condições da produção de alimentos e tomar decisões com base nos dados apresentados.
O projeto foi desenvolvido utilizando os conceitos estudados no primeiro semestre da disciplina de Computational Thinking with Python, incluindo listas, funções, estruturas de decisão, estruturas de repetição, tratamento de exceções e menu interativo.

Funcionalidades:
Consulta das plantas da estufa
Monitoramento dos sensores
Visualização de alertas
Cálculo de necessidades da missão
Informações sobre o projeto

Como executar:
1. Abrir o arquivo cosmofarm.py no Spyder ou em outro ambiente Python.
2. Executar o programa.
3. Utilizar o menu para navegar entre as funcionalidades.
